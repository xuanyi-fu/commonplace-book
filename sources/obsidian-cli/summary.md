# obsidian-cli

This source collection contains official Obsidian CLI documentation and CLI-related changelog snapshots. The main content covers CLI installation, command usage, TUI behavior, developer commands, troubleshooting, and version-specific changes.

## Structure

- `source/official-cli-overview.md`: product overview and install/getting-started material
- `source/official-help-cli.md`: command reference and troubleshooting
- `source/changelog-*.md`: CLI-specific excerpts from official release notes
- `summary.md`: the collection summary and usage guide

## How To Use

- Start with `source/official-cli-overview.md` for the product-level picture
- Use `source/official-help-cli.md` as the canonical command reference
- Use the `source/changelog-*.md` files to track when commands, behavior, or installation details changed
- Use this collection as the canonical raw source when writing wiki pages about Obsidian CLI

## Summary

This collection documents the official Obsidian CLI as a terminal interface for interacting with Obsidian. It includes installation guidance, command syntax, TUI usage, automation examples, developer tooling, platform-specific troubleshooting, and the release history of CLI-related changes.
