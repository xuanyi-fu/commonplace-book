# MCP Apps

`MCP Apps` is the official MCP extension for delivering interactive UI from MCP servers to hosts. In the official overview and stable spec, the core pattern is: tools point to predeclared `ui://` resources, hosts render those resources in sandboxed iframes, and the embedded app communicates with the host over JSON-RPC via `postMessage`. [[sources/mcp-apps-2026-04/source/official-doc-mcp-apps-overview-markdown|official-doc-mcp-apps-overview-markdown.md]] [[sources/mcp-apps-2026-04/source/official-spec-mcp-apps-2026-01-26-markdown|official-spec-mcp-apps-2026-01-26-markdown.md]]

As an official public timeline, the collected material shows MCP Apps first appearing as a proposal on `2025-11-21`, then being described as live on `2026-01-26` as the first official MCP extension. The launch material positions it as the standard way for tool results to become dashboards, forms, visualizations, and other interactive views directly inside clients such as Claude, ChatGPT, Goose, and Visual Studio Code.
